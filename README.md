# actix-web-router-example

用 Rust 的 actix-web 套件開發 HTTP Service CRUD。

## Reo

## 一、routes-example

展示 actix-web 的 http 路由各種撰寫方法。

## 二、crates-and-modules

展示 Rust 模組的匯入匯出使用。

## 三、directory-example

利用 Rust 的 crates 和 modules，將 actix-web routes 模組化。
